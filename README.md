# 📊 India General Election Results 2024 – Excel Dashboard

## 📌 Project Overview

This project presents a comprehensive analysis of the **India General Election Results 2024** using Microsoft Excel.
The dashboard provides **state-wise and constituency-wise insights** including seat distribution, winning parties, candidates, and voting patterns.

---

## 🎯 Objectives

* Analyze election results at **state and constituency level**
* Identify **majority alliance (NDA / INDIA / OTHERS)** by state
* Visualize data using **interactive Excel charts and maps**
* Provide insights for **easy understanding by end users**

---

## 📂 Dataset Details

The dataset includes:

* State Name
* Constituency Name
* Winning Candidate
* Party / Alliance (NDA / INDIA / OTHERS)
* Total Votes
* Margin of Victory

---

## ⚙️ Data Processing

* Used **Pivot Tables** for summarizing data
* Created relationships between multiple tables (Data Model)
* Built calculated columns:

  * Majority Alliance
  * Color Identifier (for visualization)
* Cleaned inconsistent state names for map visualization

---

## 📊 Dashboard Features

### 1️⃣ Overall Result Analysis

* Pie Chart showing total seats by alliance
* Quick overview of election outcome

---

### 2️⃣ State-wise Seat Distribution

* Stacked Column Chart
* Shows NDA, INDIA, OTHERS seats per state

---

### 3️⃣ Majority Alliance by State

* Map Chart (Geo visualization)
* Color-coded states:

  * 🔵 NDA
  * 🟠 INDIA
  * ⚫ OTHERS

---

### 4️⃣ Constituency-Level Analysis

* Displays:

  * Winning Candidate
  * Party
  * Total Votes
  * Margin
* Built using Pivot Table

---

## 🎨 Visualization Techniques

* Used **color coding for better readability**
* Applied **user-friendly design for quick understanding**
* Avoided complex gradients to improve clarity

---

## 🧠 Key Insights

* Easily identify which alliance dominates each state
* Compare vote margins across constituencies
* Understand national vs regional trends
